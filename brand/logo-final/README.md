# Öz & Saye Psikoloji — Final Logo Assets

Bu klasör, son onaylanan yeşil + adaçayı + soft pembe logo yönünün web'e doğrudan yüklenebilir SVG kaynaklarını içerir.

## Renk paleti

- Orman Yeşili: `#1F3B2E`
- Adaçayı Yeşili: `#A6B79B`
- Soft Pembe: `#D8A7A5`
- Sıcak Ivory: `#F5F2EB`

## Dosyalar

- `primary-vertical.svg`: ana dikey logo
- `primary-horizontal.svg`: site header / yatay kullanım
- `icon-mark.svg`: yalnız amblem
- `favicon.svg`: kare favicon / sosyal ikon kaynağı
- `reverse-horizontal.svg`: koyu zemin yatay kullanım
- `mono-forest.svg`: tek renk forest kullanım
- `mono-black.svg`: tek renk siyah kullanım

PNG çıktıları ve favicon boyutları için yerel üretim paketi ayrıca hazırlanmıştır.
