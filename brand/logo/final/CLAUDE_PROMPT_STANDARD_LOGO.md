# Claude Code Prompt — Final logo standard boyut entegrasyonu

Repo: `OmerYasirOnal/ozsaye-psikoloji`

Amaç:
Öz & Saye Psikoloji final logosunu standart, daha okunur ve profesyonel boyutta siteye entegre et.

Boyut standardı:
- Logo daha önce fazla küçük kalmıştı.
- Yeni standartta logo kare alanın yaklaşık %70–75'ini kaplamalı.
- Profil ikonlarında logo okunur olmalı; çok fazla beyaz boşluk kalmamalı.
- Open Graph/link önizleme görselinde logo genişliğin yaklaşık %60'ını kaplamalı.
- Header/Footer içinde logo taşmamalı, mobilde dengeli kalmalı.

Kullanılacak dosyalar:
- `logo_standard_1200_white.png`: ana logo.
- `logo_standard_1200_transparent.png`: transparan ana logo.
- `instagram_profile_standard_1080.jpg`: Instagram profil.
- `x_profile_standard_400.jpg`: X profil.
- `site_icon_standard_512.png`: site/app icon.
- `open_graph_standard_1200x630.jpg`: link önizleme.
- `favicon_standard_32x32.png`, `favicon_standard_48x48.png`, `favicon_standard_64x64.png`: favicon.

Renkler:
- Forest Green: `#1F3B2E`
- Sage Green: `#A6B79B`
- Soft Blush: `#D8A7A5`
- White: `#FFFFFF`
- Warm Ivory: `#F5F2EB`

Yapılacaklar:
1. `public/` içindeki eski logo, favicon ve OG görsellerini bu standart boyutlu assetlerle değiştir.
2. Header/Footer görünümünü mobil ve desktop kontrol et.
3. Instagram/X profil için standart kare versiyonları kullan.
4. `npm run lint` ve `npm run build` çalıştır.
5. Yeni branch açıp PR oluştur.
