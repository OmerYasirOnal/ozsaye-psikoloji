# Claude Code Prompt — Öz & Saye Psikoloji site-ready final logo entegrasyonu

Repo: `OmerYasirOnal/ozsaye-psikoloji`

Amaç:
Onaylanan final logonun büyük ve tüm siteye uygulanabilecek versiyonlarını projeye entegre et.

Kullanılacak dosyalar:
- `logo_big_4096_white.png`: büyük master logo.
- `logo_big_4096_transparent.png`: büyük transparan master.
- `logo_web_2048_white.png`: web ana logo.
- `logo_site_1200_white.png`: genel site içi kullanım.
- `site_header_logo_transparent_1400x420.png`: Header/Footer.
- `open_graph_1200x630.jpg`: link önizleme.
- `instagram_profile_1080.jpg`: Instagram profil.
- `x_profile_400.jpg`: X profil.
- `favicon_32x32.png`, `favicon_48x48.png`, `favicon_180x180.png`, `favicon_512x512.png`: favicon/app icon.

Renk paleti:
- Forest Green: `#1F3B2E`
- Sage Green: `#A6B79B`
- Soft Blush: `#D8A7A5`
- White: `#FFFFFF`
- Warm Ivory: `#F5F2EB`

Yapılacaklar:
1. `public/logo.png` ve ilgili logo dosyalarını final assetlerle güncelle.
2. Header/Footer’da `site_header_logo_transparent_1400x420.png` kullan.
3. `public/og.png` veya metadata görselini `open_graph_1200x630.jpg` ile değiştir.
4. Favicon ve app icon dosyalarını yeni paketle hizala.
5. Mobil/desktop görünümünü kontrol et.
6. Eski logo varyasyonlarını `brand/archive/` altına taşı.
7. `npm run lint` ve `npm run build` çalıştır.
8. Branch aç, commit at ve PR oluştur.

PR başlığı:
`Final site-ready logo assets`
